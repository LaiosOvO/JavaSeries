### 缓存(14天)
2022/9/5 -- 2022/9/19

#### 学习建议
鱼皮:
    学会如何简单地使用缓存并不难，和数据库类似，无非就是调用 API 对数据进行增删改查。

因此，建议先能够独立使用它，了解缓存的应用场景；再学习如何在 Java 中操作缓存中间件，并尝试和项目相结合，提高系统的性能。

跟着视频教程实操一遍即可，可以等到面试前再去深入了解原理和高级特性。
